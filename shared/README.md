# Shared Definitions

Shared protocol definitions and data models used by both the EDGE Server and EDGE Client.

## Structure

| Directory | Description |
|---|---|
| `protocols/` | Protocol message definitions (MQTT topic schemas, Modbus register maps, WebSocket event schemas) |
| `models/` | Shared data models (workout program structure, user profile schema, tag definitions) |

These definitions serve as the contract between server and client, ensuring consistent data structures across the system.
